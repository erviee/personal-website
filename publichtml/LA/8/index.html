Computer Programming

The process of designing and building an executable computer program to accomplish a specific computing result or to perform a specific task.

Example topics in Computer Programming

HTML

HTML is the foundation of web development. It is used to structure the content of a web page, defining elements like headings, paragraphs, lists, links, images, forms, and more. HTML uses a tag-based syntax to define the structure of a web page.

CSS

CSS is used to style and format the content created with HTML. It defines how elements should appear on the web page, including layout, colors, fonts, and spacing. You can apply CSS to HTML elements using selectors, properties, and values. For example, you can change the font color of a heading using CSS like this:

 h1 {
 color: blue;
 }

Javascript

JavaScript is a programming language that adds interactivity and dynamic behavior to web pages. It allows you to create features like image sliders, forms with validation, real-time updates, and more. JavaScript can be embedded directly in HTML pages or linked as external scripts. It can interact with the Document Object Model (DOM) to manipulate and change the content and behavior of web pages.

Keyboard shortcuts while coding

Using keyboard shortcuts while coding can greatly improve your efficiency and productivity. Most integrated development environments (IDEs) and code editors provide a wide range of keyboard shortcuts to help you navigate, edit, and debug your code more effectively. Here are some commonly used keyboard shortcuts for coding:

  1. General Navigation:
    • Ctrl + S: Save your code.
    • Ctrl + Z: Undo the last action.
    • Ctrl + Y: Switch between open files or tabs.
    • Ctrl + Tab: Undo the last action.
  2. Debugging:
    • F5: Start debugging.
    • F10: Step over.
    • F11: Step into.
    • Shift + F11: Step out.
